Question-1: What are the different ways to select an element in the DOM?

Answer : Different ways to select an element in the DOM are getElementById, getElementsByClassName, getElementsByTagName, querySelector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : innerHTML means Html inside and element , innerText means visible text inside an element and innerContent means all text inside an element.

Question-3: What is event delegation in the DOM?

Answer : Event delegation means using a single event listener to a parent element to hanlde it's children.

Question-4: What is event bubbling in the DOM?

Answer : Event bubbling is a process where event start from target element then go to uper element like parent , grand parent and continue it till it reach to document.
